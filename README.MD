
# Met'hanoï

v0.0.6

Résolvez le casse-tête des tours d'Hanoï ! Ce jeu est conçu pour la Gamebuino META.

Fonctionnalités complémentaires au jeu :

* Gestion des meilleurs scores.
* Interface en anglais et en français.
